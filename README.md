# CrimsonLedger
A dark, lightweight companion app for tracking the ever-changing state of a vampire during play: Hunger, Humanity, Health, Willpower, and other temporary session-based values.

## Crimson Ledger
Crimson Ledger is a lightweight companion app for Vampire: The Masquerade players who want to track the values that change constantly during play. Instead of storing full character sheets, it focuses on volatile session data such as Hunger, Humanity, Health, Willpower, stains, temporary conditions, and other short-term resources. Built for quick updates at the table, it keeps the most important changing stats visible, simple, and easy to manage.
